VISUAL HIERARCHY
=================

Level 1: Multilingual Panel (Green border, collapsible)
  │
  ├── Level 2: Source Language (Dropdown)
  │
  ├── Level 2: Target Languages (Grid with checkboxes)
  │
  ├── Level 2: Translation Method (Radio buttons)
  │
  └── Level 2: Voice per Language ← NEW
       │
       └── Level 3: Per-language voice rows
            ├── Language code (EN, ES, FR...)
            ├── Language name (English, Spanish, French...)
            └── Voice dropdown (Select specific voice)

  - Overflow: Auto scroll
  - Border: 1px solid gray-200
  - Border radius: 0.5rem
  - Padding: 0.75rem
  - Background: White

Language Row:
  - Flex container
  - Gap: 0.75rem
  - Padding: 0.5rem

  - Name: 8rem width, text-sm, gray-700
  - Dropdown: flex-1, focus ring green-500

Help Text:
  - Size: xs (0.75rem)

  - Style: italic
  - Margin top: 0.5rem
  - Icon: 💡 emoji

RESPONSIVE BREAKPOINTS
=======================

Desktop (≥1024px):
  - Full width within panel
  - No horizontal scroll
  - Vertical scroll after 6 languages

Tablet (768px-1023px):
  - Same layout
  - Slightly reduced padding
  - Vertical scroll after 5 languages

Mobile (<768px):
  - Stack elements
  - Full width dropdowns
  - Touch-friendly targets (44px min)
  - Vertical scroll after 3 languages

INTERACTION STATES
==================

Default:
  └─ Show if targetLanguages.length > 0

Empty:
  └─ Hidden when no languages selected

Loading:
  └─ Inherit from parent panel

Focus:
  └─ Dropdown shows 2px green ring

Hover:
  └─ Row background transitions to gray-100

Active:
  └─ Dropdown open, options visible

Disabled:
  └─ N/A (always interactive when visible)
